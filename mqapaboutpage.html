About MQAP






This project is dedicated to archiving files of Monkey Quest, Monkey Quest rewritten servers, and community servers. Once the Monkey Quest files have been organized this project will use those files to create a new Monkey Quest. All files will be open to the public as they should be. You can find these files from the websites 'Download Files' option, or, you can join the Discord. You'll be notified of any big updates to the project here on this website or in the Discord. If you have any recommendations, concerns, or assets for Monkey Quest, please contact an admin in the discord server.
